## 🔍 CAUSA RAÍZ

### Código Problemático (FIX 198):
En `agente_ventas.py`, el método `_construir_prompt_dinamico()` no recibe parámetros:

```python
def _construir_prompt_dinamico(self) -> str:
    # ... código ...

    # ❌ LÍNEA 4329: Variable respuesta_cliente NO EXISTE en este scope
    respuesta_lower = respuesta_cliente.lower() if respuesta_cliente else ""

    # ❌ LÍNEA 4408: Variable respuesta_cliente NO EXISTE en este scope
    Cliente dijo: "{respuesta_cliente}"
```

### Por Qué Sucedió:
- FIX 198 agregó validación de saludo "Dígame"
- El código asumió que `respuesta_cliente` estaría disponible
- Pero `respuesta_cliente` solo existe en `procesar_respuesta()`, NO en `_construir_prompt_dinamico()`
- Cuando GPT-4o ejecuta el método, Python lanza `NameError`
